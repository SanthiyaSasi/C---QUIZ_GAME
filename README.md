### Quiz Game Application (C++)
## Overview
This is a console-based Quiz Game developed using C++. The application presents multiple-choice questions to users, evaluates their answers, calculates scores, and provides the option to play again.
## Features
- 50 Multiple Choice Questions
- Instant Answer Validation
- Score Calculation
- Replay Option
- User-Friendly Console Interface
## Technologies Used
- C++
- Object-Oriented Programming Concepts
- Arrays
- Loops
- Conditional Statements
## How to Run
1. Clone the repository: 
   git clone <https://github.com/SanthiyaSasi/C---QUIZ_GAME>
2. Open the project in your preferred C++ IDE or compiler.
3. Compile and run the program:  
   g++ quiz_game.cpp -o quiz_game
  ./quiz_game
